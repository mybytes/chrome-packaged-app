chrome-packaged-app
===================

myBytes Chrome Packaged App


#### Quick start

* Requires http://yeoman.io/ with the AngularJs generator - to install:
   * [Install NodeJs](http://nodejs.org/)
   * `npm install -g yo`
   * `npm install -g generator-angular`

* Fetch all dependancies:
  * `bower install`

* Run dev server:
  * `grunt server`
